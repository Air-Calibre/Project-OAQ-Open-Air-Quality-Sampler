To make an OAQ Sampler of your own, you need to 3d print all the individiual stl files in the Enclosure folder, then based upon the particulate matter sensor that you are using, 3d print the holder frame from respect sub folders. 

So, if you want to use the PMS9003 Sensor within project OAQ, then 3d print a holder for the same from its respective folder (/Enclosure/PMS003_Holder. 

The assembly of Project OAQ consists of the following components. 
1. Top Cover 
2. Bottom Cover 
3. Buttons
   a. Power Button
   b. Function Button
   c. Buzzer Button
4. Sensor Holder
